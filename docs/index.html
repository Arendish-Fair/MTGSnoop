Magic Arena Snoop
Pre-Alpha
Welcome to Arenish Fair's MTGA Snoop.

This is a pre-alpha release of Magic the Gathering Arena game tracker. I wanted to create something to track my magic decks and play on my mac. Since there was nothing out there, I've created this. It will work on both windows and mac.
What it is
This is a pre-alpha release for a game tracker. Once installed, this app will show you what cards are in your library and the percent chance to draw the card. This is not a plug-in for Arena but a separate app that can read the logs. This is a test for this application to get feedback from users, and if there is any desire for this application. you can add feedback on the github site. Because this is a pre-alpha, I'm using limited server bandwidth. There may be some daily limits hit. But if this does happen, we will start to upgrade. Please give it a try.

How to Use
Once you have downloaded the app, just start it up. You will need to login using a google account to start with. We may add other ways to login in the future. once that is done then on the default screen add the file path to your Arena log dir. It will normally be at location /Users/<user>/Library/Application Support/com.wizards.mtga/Logs/Logs/ on a mac and C:\Program Files (x86)\Wizards of the Coast\MTGA) -> MTGA_Data -> Logs -> Logs on windows. Then start playing games. Once you start a game, your deck will show up under User Info. We are currently not sending any information back about the game or your deck. We have the tracker able to pick up games if you do crash. I will keep this application open until Oct 30, 2020.
